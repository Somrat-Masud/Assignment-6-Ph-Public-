Question-1: Discuss the scope of var, let, and const

Ans: var declarations are globally scoped or function scoped and let and const are block scoped
var variables can be updated and redeclared within its scope; Let variables can be updated but not redeclared; const variables cannot be updated or redeclared.


Question-2: Tell us the use cases of null and undefined

Ans: 1,Null
Null means an empty value or a blank value. The typeof() operator returns undefined for an undefined variable. The typeof() operator returns the type as an object for a variable whose value is assigned as null. 2.undefined
The undefined indicates that a variable has not been assigned a value, or declared.
